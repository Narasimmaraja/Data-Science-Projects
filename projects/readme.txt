Face Identification Project

Overview:
    This project is a real-time face identification system using Python, OpenCV, and face_recognition libraries. It captures video from the default camera, detects faces, and compares their encodings with known faces encodings to identify individuals.

Requirements
	*Python 3.x
	*OpenCV
	*face_recognition.

Background Changer Project
    This project uses the OpenCV, cvzone, TensorFlow, and Mediapipe libraries to create a real-time background changer using selfie segmentation. It allows you to replace the background of your webcam feed with images from a specified directory.

Prerequisites
    - Python 3.x
    - OpenCV
    - cvzone
    - TensorFlow
    - Mediapipe
    -SelfieSegmentation


Controls
    Esc: Exit the application
    q: Move to the previous image in the directory
    w: Move to the next image in the directory
